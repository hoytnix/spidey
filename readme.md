## Getting Started

1. `cd PROJECT_ROOT`
2. `virtualenv -p python3 ./env`
3. `source ./env/bin/activate`
4. `pip install -r requirements.txt`
5. `python run.py --help`

COPY-PASTA:

    cd /spider
    virtualenv -p python3 ./env
    source ./env/bin/activate
    pip install -r requirements.txt
    python run.py --help
